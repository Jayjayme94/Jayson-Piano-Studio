## ðŸ› ï¸ Management & Editing

### Local Preview
Simply open `index.html` in any modern web browser to view the site as it will appear online.

### Customizing Styles
Edit the `:root` variables in `style.css` to update colors or spacing globally:
```css
:root {
    --primary-color: #YOUR_COLOR;
    --radius: 12px;
}
```

### Updating Content
- **Main Page**: Edit `index.html` for hero text, about section, and lesson offerings.
- **Parents Info**: Edit `parents.html` to update office hours or studio policies.
- **Images**: Place new images in the `assets/` folder and update `src` attributes in the HTML files.
